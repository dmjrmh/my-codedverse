Latest Portfolio

Singapore National Academy

In the Singapore National Academy project, I took on the role of transforming static designs from Adobe Illustrator into a dynamic and fully functional website. The platform serves as a comprehensive digital hub for SNA, featuring essential school information such as activities, facilities, enrollment procedures, news & events, and connections to the SNA community. This project required meticulous attention to detail to ensure that the original design integrity was preserved while optimizing the site’s responsiveness, user experience, and functionality. By bridging the gap between design and development, I aimed to create a seamless, visually appealing, and intuitive platform that effectively represents the school’s identity and enhances user engagement.

Cerita Visual

Ceritavisual is a landing page development project inspired by Gatsby Ghost, designed to enhance client website engagement. This project leverages Contentful as a headless CMS for efficient content management, allowing content to be dynamically fetched via API. The HTML structure is pre-designed, ensuring smooth integration with Contentful’s API for real-time content updates. For styling, the project utilizes Bulma CSS, a modern framework that provides pre-built components and a responsive grid system. This combination streamlines the development process, allowing the team to focus on delivering a high-quality user experience without backend complexities.The result is a professional, dynamic, and visually compelling landing page that effectively boosts user engagement and strengthens the client’s digital presence.

Mom Uung

Momuung.id is a website project I developed using PHP, Bulma CSS, and JavaScript. It is a redesign of the client's previous design, provided in CorelDRAW for both mobile and desktop versions. My role was to transform the design into a fully functional and responsive website. The website features a blog, a carousel, and a product showcase that directs users to WhatsApp or the client's e-commerce platform. It also integrates Contentful as a headless CMS for managing dynamic content and Tawk.to as a chat widget for real-time customer interaction.

Sea Safari Cruises

Sea Safari Cruises is a system developed to enhance efficiency in voyage management and ticket sales for passengers traveling to destinations such as Raja Ampat, Labuan Bajo, South Halmahera, Alor, and more. This project started from scratch. I handled data migration and master data setup based on the ERD provided by the project manager to meet the client's business process requirements. In this project, I collaborated with other developers and was responsible for updating the features that had already been implemented. My contributions include: Setting up data migration and master data according to the project manager’s ERD. Adding a QR Code module to facilitate passenger attendance before departure. Developing a real-time API for exporting Excel CSV, enabling marketing agents to promote ticket sales with up-to-date data. Adding guide data for diving instructors when needed. Updating the Frappe/Gantt calendar, which helps monitor the ship departure schedule with color indicators representing different statuses: available, confirmed departure, chartered, or maintenance.

Qunci Pintar Indonesia - Finance Dashboard

QPI - Finance Dashboard is a system designed to manage the distribution of travel allowances for drivers during the delivery process by logistics companies. This system collaborates with a technology-based cooperative that provides an API for loan applications and repayments, where the funds are initially provided by the cooperative and later reimbursed by the logistics company. The system utilizes Supabase as a database to store driver information, transactions, and verification statuses from the cooperative (accepted/declined). The user interface is developed using Tailwind CSS with the Lineone template, while the backend is built with Ruby on Rails, incorporating RESTful API integration for external services

Pirate Dinner Cruises

Pirate Dinner Cruises is a management system for lunch and dinner cruises. The system helps manage trip schedules, reservations, and essential information related to vessels and passengers. In this project, my contributions include: Implementing AJAX to display trip details for specific dates in the calendar, allowing users to view real-time information without refreshing the page. Integrating QR Code functionality to streamline the check-in process and validate passengers before departure.
